How are files grouped into sequence sets?

A regex decides which files belong together. R1/R2 pairing is the common case — but a sequence set can group more than a pair (extra lanes, a reference FASTA). Pattern lives per-import, not at the lab-admin level.

Preset
Underscore _R1/_R2 Dot .R1/.R2 Illumina _S#_L###_R# Custom regex…

sample = the sequence-set id · read = which file in the set.
